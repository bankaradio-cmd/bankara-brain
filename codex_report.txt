## 1. プロジェクト概要

バンカラブレイン（Bankara Brain）は、YouTubeチャンネル「バンカラジオ」（登録者124万人）の
動画分析・脚本インテリジェンス基盤。

### 目的
- 過去60本の動画をデータ分析し、脚本制作に活かす
- 「分析 → 脚本 → 撮影 → 編集 → 投稿 → 分析」の循環を回す

### 主要な成果物
- V5深層分析: 全60本完了（各動画のシーン別分析、コメディ構造、維持率相関）
- 横断分析3本: ヒット法則20個、失敗パターン11個、維持率崩壊チェック17項目
- バンカラジオのルール: 脚本の壁打ちから確定した27の実践ルール
- キャラ別セリフ辞典 + 行動原理（計55KB）

## 3. 設計上の特徴

### 責務分離（AGENTS.md）
- **Brain（コア）**: ingest, storage, models, analytics, embeddings, feedback
- **Consumer（外部）**: 脚本生成, UI, 制作ワークフロー
- `bankara_script_assistant/` はconsumerとして分離されている

### AI制御の仕組み
- CLAUDE.mdで「AIの暴走パターン」を経験ベースで列挙して制止
- 例: 「勝手に次の作業に着手しない」「シーンをまとめない」「制作手法の推測禁止」
- 脚本壁打ちのフロー: context pack読み込み → ゲート通過 → 構成案 → FBループ
- フィードバックの昇格は人間（やねすけ）の判断制。AIが勝手に昇格しない

### データの流れ
```
YouTube API → bankara_brain.db → V5深層分析 → 横断分析
                                      ↓
                              脚本壁打ちで活用
                                      ↓
                              公開後フィードバック（未実装）
```
