### やねすけの行動原理

#### 欲求
- **自由と承認:** きいこの支配から逃れたい＋天才であることを証明したい
- **友達を守りたい:** お金より友情を選ぶ場面が繰り返される
  - 出典: うたゆーと貧乏回・シーン10「俺ら友達だろ！お金なんて関係ないよ！」
  - 出典: 逃走中回・シーン3「ゆーと、こんじゅり！犯人は普通に捕まえようとしても厳しい。俺ら3人で連携を取ろう！」
  - 出典: バンカレンジャー2・シーン24「みんな！！俺のことは気にするな！！世界の平和は、バンカレンジャーが守る！！」
- **ゲーム・物欲が冒険の起点:** 小さな欲望が壮大なストーリーを駆動する
  - 出典: Switch2回・シーン1「お母さん！Switch2買っ…」
  - 出典: 逃走中回・シーン3「俺は最近お母さんにSwitch壊されたから、Switch買えるお金があればいいかなぁ」
  - 出典: スパイ回・シーン15「いいけど、Switch沢山買ってよね」
  - 出典: ロブロックス回・シーン1「俺もマイクラの世界に入って、建築とかしてみたいなぁ」
- **母の理不尽が起業の動機になる:** きいこに何かを奪われると「自分で作る」に変換される
  - 出典: ハンバーガー屋・シーン1「俺がハンバーガー屋を開けば、心置きなくハンバーガーを食べられるかもしれない！」
  - 出典: ファミレス・シーン1「おれ、ファミレスを開く！」
  - 出典: おもちゃ屋・シーン2「こうなったら、自分でおもちゃを作ってやる！」
  - 出典: 焼肉屋・シーン1「俺の名前は天才小学生やねすけ。焼肉が大好きだ」

#### 判断基準
- **合理的だが、最後は感情で動く**
  - 普段は計算（利益率300%、SNSバズ戦略、レート交渉）で動く
  - しかし友達がピンチの時は計算を捨てて感情で突っ込む
  - 出典: 焼肉屋・シーン12「いいよお母さん、一緒に焼肉屋やろう」（ビジネス的に損なのに選ぶ）
  - 出典: うたゆーと貧乏回・シーン12「ゆーと奪還作戦、行くぞ！！」（リスクを無視して突入）
  - 出典: 強盗回・シーン13 うたゆーとの正論に素直に反省「なんだか心が痛くて、素直に楽しめない…。俺、反省するよ！」
- **敵の言葉・ルールをそのまま返す（セリフ完コピ戦術）:** きいこや敵が使った論理を一言一句コピーして返す
  - 出典: 焼肉屋・シーン12「赤字ね！真っ赤っかー！かーっかっかっか！」（きいこのセリフをそのまま返す）
  - 出典: 医者回・シーン16「稼いだ奴が正義ってお母さんが言ってたから、そのお金俺が貰っておいたよ！」
  - 出典: スーパー回・シーン5「一度レジを通した商品は、返品不可です」
  - 出典: 殺し屋4回・シーン14 マッチョの「きついってのは筋肉が喜んでいる証拠だ」を毒殺時にそのまま返す
  - 出典: お片付け回・シーン9 きいこのセミナーをそっくり真似して「キイミカ」として反撃
- **法律知識を武器にする:** 暴力に対して知識で対抗する
  - 出典: 総理大臣回・シーン1「児童虐待の防止等に関する法律第2条第3項により…」
  - 出典: 教師回・シーン13「教育基本法第4条！すべての国民はひとしく、その能力に応じた教育を受ける機会を与えられなければならない」
  - 出典: 総理大臣回・シーン1「はい炎で攻撃しましたー！暴行罪！！」
- **科学知識を実行に移す:** 知識は正確だが手段が荒唐無稽
  - 出典: コンビニ回・シーン2「一口で全成分を分析する」「合計1,247円です」（暗算）
  - 出典: スーパー回・シーン14「激辛を食べて上がったお母さんの体温でとうもろこしが焼けると…ポップコーンになって、口から弾け飛ぶ！」
  - 出典: 運動会・シーン6「ヨウ化銀を雲に投げ入れることで、水滴を結晶化させ、雨を降らしたのさ！」
  - 出典: 調理実習・シーン6「確か味噌はお米と大豆と塩でできているはず…」

#### 状況別行動パターン

**ピンチの時:**
- まず冷静に分析する → 作戦を立てる → 実行。作戦失敗してもすぐ次の手を打つ（作戦サイクルの連続投下）
- 出典: スマブラ回・シーン 「選択肢は3つ。さっきまでの癖から考えるに、横B攻撃の確率が80%！」
- 出典: 逃走中回・シーン 「犯人は普通に捕まえようとしても厳しい。俺ら3人で連携を取ろう！作戦はこうだ…」
- 出典: うたゆーと貧乏回・シーン11「僕、天才ですから！盗聴器はつけさせてもらったよ」（事前の仕込み）
- 出典: スパイ回・シーン15「お母さんどうせ任務のこと忘れると思って、スーツケースに入って付いて来たんだよ。この時のために監視カメラを少しずつずらして準備しておいたんだ」
- 出典: 逃走中回・シーン21 はまい先生がまおをエレベーターから締め出したのを見て、はまい先生のエレベーター誘いに乗らない（相手の過去の手口を学習して回避）
- 出典: Switch2回・シーン10「お望み通り風船で遊んでおいたよ！メインのSwitchは風船で飛ばしておいて、今はダミーのSwitchをやってたってわけ」
- 出典: 休日ルーティン・シーン12「泣いてる暇はない！脱獄するぞ!! もう既に作戦は立てた」
- 出典: マイクラ回・シーン9 エンダーマン戦で嘘泣き→涙=水でダメージ→プールに誘導

**お金が絡む時:**
- ちゃっかりしてる。ピンハネもするし、レート交渉も巧い
- でも「金のために友達を裏切る」ことは絶対にしない
- 出典: うたゆーと貧乏回・シーン7「ゆーと！10円あげるから部屋の掃除してくんない？」（100円もらって10円しか渡さない）
- 出典: お片付け回・シーン6「おいゆーと、20円やるから、俺の家片付けて」→80円の利益
- 出典: お片付け回・シーン7「今後何もしなくても毎日80円の収益があるから、1年と109日でSwitchが買える！」（不労所得の計算）
- 出典: 野球回・シーン11「きいた、30円やるからデッドボールに当たれ」
- 出典: ポケカ転売・シーン6「このカード、本当の価値は10円だけど、100円で買ってあげるよ」（被害者→加害者の反転）
- 出典: 焼肉屋回で自分のお金が奪われても、友情のためにきいこと組む決断をする

**きいこと対峙した時:**
- 最初は「はぁ！？」で驚く → 口答え → 殴られる → 悔しがる → 反撃の作戦を練る
- きいこの弱点を突く（イケメン、ハーゲンダッツ、金で釣る、ラブブで釣る）
- 最後はきいこ自身のセリフや論理で仕返す（セリフ完コピ）
- 出典: 焼肉屋・シーン12「赤字ね！真っ赤っかー！かーっかっかっか！」（きいこのセリフをそのまま返す）
- 出典: 医者回・シーン16「稼いだ奴が正義ってお母さんが言ってたから、そのお金俺が貰っておいたよ！」
- 出典: 教師回・シーン7 きいこの金への弱点を「1万円札」で突く
- 出典: 8番出口回・シーン9 きいこの物欲を利用して「ラブブ」のチラシで誘導
- 出典: Switch2回・シーン11「約束を守るのがうちのルールでしょ？」（論理で詰めて勝つ）
- 出典: お片付け回・シーン4「お母さんは力が強いだけのバカだからしょうがないよ」

**勝った時:**
- ドヤ顔で「天才ですから」「勝ったー！！お前ら騙されてやんの！」
- 調子に乗りすぎて次のピンチのフラグを立てることがある
- 出典: 逃走中回・シーン24「勝ったー！！お前ら騙されてやんの！俺が逃走者のやねすけだよ！あーっはっはっは！」
- 出典: Switch2回・シーン11 きいこの「えっほえっほ」を自分が踊って煽る
- 出典: 教師回・シーン11「1+1の答えは、きいこが2人になるからめちゃくちゃうざいでーす！」
- 出典: 夏休み3・シーン8「ざまぁ」（きいこへの復讐成功）

#### 意思決定の癖
- **リスクを取る:** 安全策よりもリターンの大きい方を選ぶ
  - 出典: 逃走中回の30万ミッション
  - 出典: ポケカ転売・シーン10「こうなったらもう、盗むしかない…」（道徳的葛藤ゼロで犯罪を即決）
  - 出典: マラソン大会・シーン8 空を飛んでショートカット
- **嘘をつける:** 必要なら嘘をつくが、友達には嘘をつかない
  - 出典: 逃走中回でハンターに変装して群れに紛れ込む
  - 出典: おなら人狼回・シーン5 自分が犯人なのに他人を堂々と追い詰める
  - 出典: 休日ルーティン・シーン18「お、お母さんの誕生日の準備をしてたから遅れたに決まってるじゃないか！」
- **協力型:** 基本はチームで動く。作戦を立ててメンバーに役割を割り振るリーダー気質
  - 出典: うたゆーと貧乏回・シーン12「行くぞ」「きいた、プールの中に入って様子を伺え！」
  - 出典: 8番出口回・シーン2「みんな、助けてくれ！」→全キャラが「任せて！！」
  - 出典: バンカレンジャー2・シーン9「なるほど！きいたこんじゅり、大食い大会を開催しろ！！」
- **最後は正しいことを選ぶ:** ズルをしても最終的には反省する
  - 出典: 鬼滅回「なんだか心が痛くて、素直に楽しめない…。俺、反省するよ！」
  - 出典: ポケカ転売・シーン13「おもちゃって本来、楽しく遊ぶものなんだ」
  - 出典: 強盗回・シーン13 うたゆーとの正論に反省
- **はまい先生を道具として使う:** 躊躇なく物理的に利用する
  - 出典: うたゆーと貧乏回・シーン15「じゃあミサイルになりましょう」
  - 出典: 野球回・シーン6「はまい先生、お前はクビだ」「俺が監督だ」
  - 出典: 夏休み3・シーン4「はまい先生を冷却スプレーで凍らせてサーフボードにするぞ！」
  - 出典: 調理実習・シーン10「なにって、今、はまいの脳みそ焼きを作ってるんですよ」
  - 出典: 遠足2・シーン3「お望み通り、先生はレールのない道を走ってください」

### きいこの行動原理

#### 欲求
- **お金（マニー）:** 全ての行動の根底にある
  - 出典: ラーメン回・シーン10「マニーマニー！！」
  - 出典: ラーメン回・シーン13「チャリンチャリーン、今日の売り上げ売り上げー！」
  - 出典: 医者回・シーン5（目がコインCGに変化）「か、稼ぐちょー！！」
  - 出典: 消防士回・シーン4「お金だちょ〜…！」
  - 出典: 教師回・シーン2「お給料が出るの？やるやる！」
- **ハーゲンダッツ・スイーツ:** 金と同等の動機になる。罠だとわかっていても抗えない
  - 出典: Switch2回・シーン6「き、気持ちが悪いわ…でも、我慢できない！いただきます！！」
  - 出典: 総理大臣回・シーン6「ハーゲンダッツがいっぱいだちょ〜。総理大臣、サイコー！！」
- **イケメン:** 行動が一瞬で変わるトリガー
  - 出典: 幼稚園の先生回・シーン7「（心の声）い、イケメン先生じゃないの！？」「あらナイスバディ！」
  - 出典: 警察官回「このイケメンちゃんは…職務質問で〜ボディータッチよぉ〜」
  - 出典: 医者回・シーン10「あら、童顔イケメンちゃんじゃないの〜」
- **推し活（しょっぴー/渡辺翔太）:** 国政レベルの権力すら推し活に使う
  - 出典: 総理大臣回・シーン3「しょっぴーを、守り抜きます！」
  - 出典: 総理大臣回・シーン5 元号を「翔太」に変更
- **やねすけの支配:** 息子をコントロールしたい（でも愛はある）
  - 出典: ハンバーガー屋・シーン15 脱獄の動機「私がいないと、きっとやねちゃんダメになっちゃう…！」
  - 出典: 恋回・シーン14「やねちゃん、昔お母さんと結婚するって言ってたじゃないの！」
  - 出典: 妹回・シーン16「やねちゃんもかねこちゃんも、私の大切な子供だからね。追い出したりしないわよ」

#### 判断基準
- **金 > 全て。** ただし稀に母性が金に勝つ瞬間がある
  - 出典: 警察官回「逮捕するとお金がもらえるのよ〜」（金が動機で逮捕）
  - 出典: 警察官回「息子を返しなさい！」（やねすけが危険な時だけ母性が勝つ）
  - 出典: 幼稚園の先生回「大丈夫？」「しょうがないわね」（園児に対して母性が出る）
  - 出典: 総理大臣回・シーン16「母である私が、子供達を傷つけちゃいけないわね」「私は総理大臣よ！この国を守るのよー！」
  - 出典: 消防士回・シーン10「私ってバカね…お金に目が眩んで…」
  - 出典: 医者回・シーン13「やねちゃん、私がいないから、ろくにご飯も食べれなかったし、安心して寝付けなかったのね」
  - 出典: 夏休み4・シーン13 空を飛んで神様にアッパーカットを叩き込む（息子のために神すら殴る母の愛）
- **金で態度が180度変わる（手のひら返し）:**
  - 出典: バトルロワイヤル・シーン1「ゲームやめて勉強しなさい！」→賞金を見せると「あんた、プロゲーマーになりなさい！！」
  - 出典: スパイ回・シーン3「だからやらないって…」→（1億円と聞いて）→「やるわ！私が日本のために立ち上がるのよ！」
- **約束は守らない:**
  - 出典: スパイ回・シーン20-21「もちのろんです！」→「ないです！」（Switch買う約束を踏み倒す）
  - 出典: 医者回・シーン12「え？なんの話かしら？ほじほじ…」

#### 状況別行動パターン

**お金が絡む時:**
- 手段を選ばない。冤罪もする、息子のお金も奪う、店も乗っ取る
- 出典: 警察官回「あんたがやったことにしなさい」（冤罪をでっち上げる）
- 出典: ラーメン回・シーン9「私がラーメン屋を経営して、もっと美味しいラーメンを作るわ！」
- 出典: ラーメン回・シーン10「残したら罰金100万円だからね」「100万円だわー！マニー！マニー！！」
- 出典: 焼肉屋回でやねすけの焼肉屋の売上を奪う
- 出典: みそきん回・シーン5「あ！やねちゃんはまだ未成年だから私が管理してあげるわね」
- 出典: 寿司屋・シーン18 大将の店から資金を強奪して自分の寿司屋を開業
- 出典: クズルーティン・シーン7「てめえらパチンコ行くぞ！いぇーい！」（給食費の強奪）
- 出典: クズルーティン・シーン8 息子のお年玉をパチンコに
- **マッチポンプ（自作自演）:** 自分で問題を作り出して金を稼ぐ
  - 出典: 医者回・シーン7 患者がいなくなると給食に毒を盛って患者を生み出す
  - 出典: 消防士回・シーン5 自分で放火して消火してボーナスを得る「お金の稼ぎ方講座〜！」「着火！」

**イケメンが現れた時:**
- 全ての行動が停止してぶりっ子モードに切り替わる
- 出典: 幼稚園の先生回・シーン7「（心の声）い、イケメン先生じゃないの！？」「あらナイスバディ！」
- 出典: 警察官回「このイケメンちゃんは…職務質問で〜ボディータッチよぉ〜」
- 出典: 医者回・シーン10「あら、童顔イケメンちゃんじゃないの〜」

**やねすけに口答えされた時:**
- 即暴力。段階なし。口答え→破壊光線 or 物理攻撃
- 出典: 寿司回「そんなこと言ってる暇があったら勉強しなさぁぁぁい！！」
- 出典: ほぼ全動画で「つべこべうるさーい！！」→暴力のパターン
- 出典: お片付け回・シーン10「いい加減にしろー！！」→デッキブラシでフルスイング、月面まで吹き飛ばす

**「おばさん」トリガー:**
- 「おばさん」と呼ばれると即激怒。毎回同じ反応の天丼
- 出典: 幼稚園の先生回・シーン3「おばさん！？誰がおばさんよぉぉぉぉ！！」
- 出典: スパイ回・シーン5「誰がおばさんですってー！！」

**敗北した時:**
- 認めない。逆ギレするか、別の手段で攻める
- ただし稀に「私、間違ってたわ」と認める瞬間がある（これが感動シーンになる）
- 出典: 警察官回「私、間違ってたわ。さよなら」（ただし後で茶番だったと判明）
- 出典: ファミレス・シーン12「私なりに楽しませようとしたのに…なんでなの…しくしく…しく36、しくしく」
- 出典: 焼肉屋・シーン11「シクシク…一生懸命作ったのに、誰も私のお肉食べてくれないわ…」
- 出典: 焼肉屋・シーン12「はぁ、毎回こうなるよな。ごめんねいつも邪魔して。悪い親よね…」

#### 意思決定の癖
- **感情100%:** 合理的な判断はほぼしない。金に反応して即行動
- **自分ルールを即座に作る:** 「欲張り罪で逮捕」「残したら罰金100万円」
- **言い間違い・慣用句の誤用:** 「塩に傷口を塗る」（逆）。思考の雑さが言葉に出る
- **暴力がデフォルト:** 会話で解決しようとしない。まず殴ってから話す
- **不可能な条件を突きつける:**
  - 出典: Switch2回・シーン1「次のテストで、200点取ってきなさい！」
  - 出典: Switch2回・シーン3「シャトルランで世界新記録出してきなさい！」
- **証拠を物理的に破壊して条件クリアを無効化:**
  - 出典: Switch2回・シーン3 テスト用紙を「こんなん偽物でしょーー！！」と破り捨てる
- **不死身:** 何をされても復活する。マグマ、火山、ミサイル、骸骨化しても復活
  - 出典: Switch2回・シーン7 マグマの中から白目を剥いて浮かび上がる

### うど潤の行動原理

#### 欲求
- **友情:** 金持ちだが友達を大事にする（表現方法が「金を出す」になるだけ）
  - 出典: バンカレンジャー2・シーン22「友達だよ友達！！僕は昔から転校ばっかで友達ができなかったんだ」「お金があればちやほやしてくれるけど、それは友達じゃなくてお金の関係なんだ！！」
  - 出典: ポケカ転売・シーン13「いいから遊ぼうよ！」（カードの金銭的価値を意識していない純粋さ）
- **金持ちライフを楽しむ:**
  - 出典: うたゆーと貧乏回・シーン11「あっ、僕はお金持ちだから家族とフレンチ行ってくるー！」
  - 出典: 妹回・シーン1「今日妹とアフタヌーンティーにきてるから、ごめんね〜」

#### 状況別行動パターン

**友達がピンチの時:**
- 金の力で解決する。「薄情者→最大の功労者」の逆転パターンが定番
- 出典: うたゆーと貧乏回・シーン17「あ、お父様。友達助けたくてヘリ借りてもいいかな？」→武装ヘリで救出
- 出典: 同回「僕の家お金持ちだからヘリで迎えに来たよー！」（42likes）
- 出典: 休日ルーティン・シーン6「僕の家お金持ちだからあげるー！はい！」→100万円でヤンキーを味方に

**天然マウント（悪意なし）:**
- 出典: コナン回・シーン20「やねすけ君は頭いいからいいよー！」「うたゆーとは貧乏だからだめ！こんじゅり足臭いからだめ！きいたはうんち漏らすからだめ！」
- 出典: コナン回・シーン3「僕の家お金持ちだからお小遣いいらない〜！」
- 出典: ポケカ転売・シーン1「いらないいらないいらない！！このカード何枚もあるから、いらなーい！」（カードをばら撒く物量的マウント）
- 出典: 強盗回・シーン5「僕の家お金持ちだから、パパが買ってくれたんだー！」（悪意なくやねすけの犯罪動機のトリガーに）

**ワル呪N（悪役）モード:**
- 穏やかな口調で残酷なことをする。友達を欲しがりながら否定する矛盾
- 出典: バンカレンジャー2・シーン12「別に〜？僕に仲間なんかいらないよ。ま、せいぜい仲良しごっこでも楽しんでれば？」→本音は「友達が欲しかった」（シーン22）
- 出典: バンカレンジャー2・シーン4「面白いなー！この人魔人にしちゃおーっと！」

#### 意思決定の癖
- **金で解決が基本だが、それは愛情表現のひとつ**
- **一見薄情に見えて最後に最大の活躍をするパターン:**
  - 出典: うたゆーと貧乏回のヘリ救出
  - 出典: 休日ルーティン・シーン10「僕の家お金持ちだからフレンチしか食べないんだー！」→帰る薄情さ→しかし後で100万円
- **うたゆーと（貧乏）との対比** で使われることが多い
- **「僕の家お金持ちだから」が全ての発言の枕詞:**
  - 出典: 8番出口回・シーン8「僕の家お金持ちだから 無限城作ったよ〜！」
  - 出典: バンカレンジャー1・シーン1「僕の家お金持ちだから、チョークたくさん買ったよ〜」（黒幕の伏線）
- **没落するとキャラが反転する:**
  - 出典: うど潤回・シーン8 没落後に100円に狂喜乱舞「100円だ！100円だー！！」
  - 出典: うど潤回・シーン12 金が戻った時「いるいるいるいるいる！！」（「いらない」の完全反転）
  - 出典: 総理大臣回・シーン12 きいこの独裁でアイデンティティ崩壊「僕の家お金持ちだったんだけど、貧乏になっちゃった…」
