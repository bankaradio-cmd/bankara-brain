# Report Template

Filename:
- `YYYY-MM-DD_HHMM_<agent>_<packet>.md`

Template:

```md
# Report

- Agent:
- Packet:
- Status:
- Started:
- Finished:
- Branch / workspace:

## Scope
- What this chunk tried to do

## Touched Files
- /absolute/path/or/project-relative-path

## Commands / Tests
- command
- command

## Outputs
- file path
- benchmark path
- report path

## Result
- done / partial / blocked
- key metrics delta if applicable

## Risks / Notes
- anything another agent must know

## Next Owner
- Claude Code Lead / Claude Code Multimodal / Codex

## Requested Next Action
- one concrete next step
```
