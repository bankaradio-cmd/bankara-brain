# Reports

Every packet handoff writes a markdown report here.

Naming:
- `YYYY-MM-DD_HHMM_<agent>_<packet>.md`

Examples:
- `2026-03-13_0400_codex_P0-CROSS-ENCODER-RERANK.md`
- `2026-03-13_0515_claudelead_P0-FEEDBACK-V2.md`

Rules:
- one report per meaningful handoff
- do not overwrite previous reports
- always include the exact next owner
