# Instagram次ステップ実行計画

更新日: 2026-03-11 JST

## 1. 結論

現時点で、**汎用リサーチは十分**です。

これ以上の一般調査を続けても、戦略の質は少ししか上がりません。むしろ次に重要なのは、以下の3点です。

1. 最初に攻めるジャンルを確定する
2. 最初に売るものを確定する
3. 対象アカウントで実際に使える収益化面を確認する

つまり、今後のボトルネックは「情報不足」ではなく、**実行前提の意思決定不足**です。

## 2. リサーチが十分と判断する理由

すでに以下は押さえています。

- Instagram/Metaの主要な収益化手段
- アフィリエイトとブランド案件の公式開示ルール
- Content Monetization Policies上のAI動画リスク
- 日本のステマ規制
- 初期に狙うべき収益モデルの優先順位
- 最初の1アカウントに向くジャンル候補
- 90日で最初の売上を作るための設計

これで、事業設計として必要な「大枠」は揃っています。

## 3. まだ残っている不確実性

ここから先は、公開情報を読んでも解けないものが多いです。

### アカウント固有の不確実性

- 対象アカウントで `Subscriptions` が有効か
- 対象アカウントで `Gifts` が有効か
- `Creator Marketplace` が表示されるか
- `Product tagging / Shop` が有効化可能か
- `Trial Reels` が使えるか

### 商材固有の不確実性

- 何を最初に売るか
- 自社商品から始めるか、アフィリエイトから始めるか
- どの価格帯が最も売りやすいか
- LINE、メルマガ、DM、予約のどれが主導線か

### 運用固有の不確実性

- どのフックが最もプロフィール遷移を生むか
- どのCTAが最もクリックを生むか
- どのフォーマットが最も保存/シェアされるか

## 4. 次にやるべきこと

次は調査ではなく、**実行可能な仮説を固定するフェーズ**です。

優先順:

1. 収益モデルを1つに絞る
2. ジャンルを1つに絞る
3. 初期オファーを1つ作る
4. プロフィール導線を1本に絞る
5. 10〜20本の検証用Reels企画を作る

## 5. 推奨する初期意思決定

現時点の推奨は以下です。

### ジャンル

`AIツール活用` もしくは `業務効率化 / 自動化`

### 初期収益モデル

`アフィリエイト + 無料リード獲得`

### 初期オファー

- 無料テンプレ
- 無料チェックリスト
- 無料PDF
- 無料診断

### 回収導線

`プロフィールリンク -> LINE or メルマガ`

### 本命の後続商品

- テンプレ販売
- ノウハウ販売
- 導入支援
- コンサル

## 6. Phase 1 実行計画

### Step 1: 収益テーマの確定

決めること:

- 誰向けのアカウントか
- その人の何の悩みを解くか
- 何を売るか

成果物:

- ターゲット定義
- 提供価値1文
- 売るもの一覧

### Step 2: 導線の確定

決めること:

- プロフ文
- リンク先
- 無料オファー
- CTA

成果物:

- プロフィール草案
- リンク先構造
- CTAテンプレ3種

### Step 3: コンテンツ仮説の確定

決めること:

- どのフォーマットで投稿するか
- どのテーマで検証するか
- 何を勝ちとするか

成果物:

- 投稿フォーマット3種
- 20本分の企画リスト
- KPI定義

### Step 4: アカウント機能確認

確認すること:

- Professional account種別
- Monetization status
- Account Status
- Gifts / Subscriptions / Marketplace / Shop / Trial Reelsの表示有無

成果物:

- 機能可否一覧

### Step 5: MVP設計

作るもの:

- 投稿ネタ管理
- 台本生成
- 動画生成
- 予約投稿
- インサイト回収
- CTA別成績比較

成果物:

- システム要件
- DB項目
- 投稿運用フロー

## 7. 実務での判断基準

次の状態になったら、追加の大規模リサーチではなく実装に入るべきです。

- ジャンルが決まっている

- CTAが決まっている
- プロフィール導線が決まっている
- 検証用の投稿テーマが並んでいる

ここまで決まれば、足りない情報は実験で取った方が速いです。
