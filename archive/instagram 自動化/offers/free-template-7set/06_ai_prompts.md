# AI Prompts

## 1. 投稿ネタ出し

```text
Instagramで集客したい小規模事業者向けに、AIでInstagram運用を効率化するテーマのリール企画を20本考えてください。
各企画に対して、悩み、フック、CTA、狙う反応を1つずつ付けてください。
```

## 2. 台本作成

```text
次のテーマを30秒のInstagramリール台本にしてください。
テーマ:
ターゲット:
主目的:
CTA:

構成は Hook -> Problem -> Solution -> Result -> CTA。
各パートは短く、字幕にしやすい日本語で書いてください。
```

## 3. フック改善

```text
次のリール冒頭を、Instagramで3秒以内に止まりやすい表現に5パターン書き換えてください。
条件:
- 誇大表現は禁止
- 具体性を上げる
- 1文目だけで悩みが伝わるようにする
```

## 4. CTA生成

```text
次の投稿内容に合うCTAを10個作ってください。
目的は1つだけにしてください。
選択肢:
- プロフィール訪問
- 無料テンプレ登録
- ツール比較クリック
- DM相談
```

## 5. ストーリー化

```text
次のリール内容を3枚のInstagramストーリーに変換してください。
1枚目は共感、2枚目は具体例、3枚目はCTAにしてください。
```

## 6. 改善レビュー

```text
次の投稿結果を見て、次に変えるべき要素を3つ提案してください。
データ:
- 再生数:
- 保存率:
- シェア率:
- プロフィール訪問率:
- リンククリック率:
```
