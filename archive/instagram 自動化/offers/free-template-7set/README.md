# AIでInstagram運用を効率化する無料テンプレ7点セット

更新日: 2026-03-11 JST

## 内容

1. `01_reel_script_template.md`
2. `02_hook_examples.csv`
3. `03_cta_templates.csv`
4. `04_story_ideas.csv`
5. `05_post_calendar.csv`
6. `06_ai_prompts.md`
7. `07_improvement_checklist.md`

## 使い方

1. `05_post_calendar.csv` で7日分のテーマを決める
2. `02_hook_examples.csv` から冒頭の型を選ぶ
3. `01_reel_script_template.md` に当てはめる
4. `06_ai_prompts.md` の指示文で台本を広げる
5. `03_cta_templates.csv` から投稿目的に合うCTAを入れる
6. 投稿前に `07_improvement_checklist.md` で確認する
7. Storiesは `04_story_ideas.csv` を使って補足する

## 想定読者

- Instagramを集客に使いたい小規模事業者
- 1人で運用している人
- AIを使って投稿作成を時短したい人
