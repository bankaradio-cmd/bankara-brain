# Reel Script Template

## 目的

- Reach
- Profile Visit
- Lead
- Affiliate Click
- DM

## 基本構成

### 1. Hook

- 誰の悩みか
- 何が解決するか
- 強い一言

記入例:

`投稿ネタが続かない人、まずこれを変えてください`

### 2. Problem

- 今のやり方の問題
- よくある失敗

記入欄:

- 問題1:
- 問題2:

### 3. Solution

- AIでどう解決するか
- 何を入力するか
- どんな出力を使うか

記入欄:

- 使うAI:
- 入力内容:
- 出力内容:

### 4. Result

- 何が短くなったか
- 何が楽になったか
- 何が売上につながるか

記入欄:

- 時短:
- 効果:

### 5. CTA

- 何をしてほしいかを1つだけ書く

記入欄:

- CTA:

## 30秒版の流れ

1. Hook 3秒
2. Problem 5秒
3. Solution 12秒
4. Result 5秒
5. CTA 5秒

## 字幕メモ

- 1画面1メッセージ
- 長文を避ける
- 数字か比較を入れる
